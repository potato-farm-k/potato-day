# Potato’s Day

감자의 하루를 그리는 조용하고 귀여운 힐링 게임 프로젝트입니다.

## 현재 버전

v0.4.2

## 변경 내용

### v0.4.2

- 감자 캐릭터를 더 포메라니안다운 전신 실루엣의 SVG 구조로 재설계했습니다.
- 실제 감자 reference와 `workflow-pom-style.png`를 참고해 아이보리 털, 진한 눈, 작은 코, 분홍 혀, 말린 꼬리 표현을 보완했습니다.
- 쿠션을 약하게 처리하고 러그 대비를 조정해 캐릭터가 배경에서 더 잘 분리되도록 했습니다.
- reference 이미지가 공개되지 않도록 `assets/reference/`를 `.gitignore`에 추가했습니다.

### v0.4.1

- GitHub Pages 배포 반영 여부를 확인하기 위한 `v0.4.1` 버전 표시를 추가했습니다.
- 감자 캐릭터의 둥근 머리, 작은 귀, 복슬복슬한 외곽, 말린 꼬리 표현을 보완했습니다.
- 감자 캐릭터가 러그와 쿠션에서 더 잘 구분되도록 외곽선, 그림자, 색상 대비를 조정했습니다.

### v0.4

- `감자의 집` 화면에 아이보리색 포메라니안 감자 캐릭터를 추가했습니다.
- 큰 눈, 작은 코, 웃는 입, 분홍 혀, 복슬한 털, 말린 꼬리 특징을 HTML/CSS로 표현했습니다.
- 기존 휴식 공간, 워크플로우 버튼, 모달 기능을 유지했습니다.

### v0.3

- 시작 화면을 `감자의 집` 허브 공간처럼 보이도록 개선했습니다.
- 창문, 러그, 쿠션, 작은 바닥 영역 등 단순한 2D 장식 요소를 추가했습니다.
- 현재 감자 상태 문구를 추가했습니다.

### v0.2.1

- 워크플로우 보기 버튼을 시작 화면에 어울리는 작은 아이콘 버튼으로 수정했습니다.
- 워크플로우 다이어그램 이미지 경로와 repository 포함 여부를 확인했습니다.
- 모달 이미지가 화면을 넘치지 않도록 표시 스타일을 보완했습니다.

### v0.2

- 시작 화면에 `루틴 보기` 버튼을 추가했습니다.
- `assets/images/workflow-diagram.png` 워크플로우 다이어그램을 모달로 확인할 수 있습니다.

### v0.1

- GitHub Pages에서 실행 가능한 기본 정적 페이지 구조를 만들었습니다.
- 시작 화면에 `Potato’s Day`, `감자의 하루`, `감자의 집` 문구를 표시했습니다.

## 실행 방법

1. 이 저장소를 로컬에 내려받습니다.
2. `index.html` 파일을 브라우저에서 엽니다.
3. 화면에 `v0.4.2`, `Potato’s Day`, `감자의 하루`, `감자의 집` 문구가 보이는지 확인합니다.
4. 아이보리색 복슬복슬한 감자 캐릭터가 러그와 배경에서 잘 구분되는지 확인합니다.
5. 큰 눈, 작은 코, 웃는 입, 분홍 혀, 작은 귀, 말린 꼬리와 전신 실루엣이 보이는지 확인합니다.
6. 시작 패널의 작은 아이콘 버튼을 클릭해 워크플로우 모달이 열리는지 확인합니다.
7. 닫기 버튼 또는 모달 바깥 배경을 클릭해 모달이 닫히는지 확인합니다.
8. 브라우저 개발자 도구 Console에서 `Potato’s Day v0.4.2` 메시지를 확인합니다.
