# Potato’s Day

감자의 하루를 그리는 조용하고 귀여운 힐링 게임 프로젝트입니다.

## 현재 버전

v0.4.4

## 변경 내용

### v0.4.4

- 홈 화면의 감자 캐릭터 이미지를 최종 투명 배경 에셋 `assets/characters/gamja-base.png`로 교체했습니다.
- v0.4.3-preview.1에서 조정한 감자 이미지 크기와 위치를 유지했습니다.
- 기존 워크플로우 보기 버튼과 모달 동작은 그대로 유지했습니다.

### v0.4.3-preview.1

- 홈 화면의 공식 감자 기준 이미지 preview 크기를 줄였습니다.
- 감자 이미지가 러그 위에 더 자연스럽게 서 있는 느낌이 나도록 위치를 살짝 아래로 조정했습니다.
- 기존 워크플로우 보기 버튼과 모달 동작은 그대로 유지했습니다.

### v0.4.3-preview

- 공식 감자 기준 이미지 `assets/characters/gamja-base-source.png`를 홈 화면의 메인 캐릭터 preview로 배치했습니다.
- 기존 코드 기반 감자 SVG 대신 독립된 이미지 레이어를 사용해, 나중에 최종 투명 PNG로 쉽게 교체할 수 있게 했습니다.
- 이번 preview 이미지는 배경 제거 전 기준 이미지이므로 체크무늬나 배경이 보일 수 있습니다.
- 워크플로우 보기 버튼과 모달 동작은 기존대로 유지했습니다.

### v0.4.2.1

- GitHub Pages에서 감자 SVG가 CSS 캐시 또는 class 스타일 누락 시 검은 실루엣으로 보이는 문제를 수정했습니다.
- 감자 SVG 내부에 아이보리/연크림색 `fill`/`stroke` fallback을 추가해 눈, 코, 입, 혀, 몸통, 꼬리 디테일이 안정적으로 표시되도록 했습니다.
- `style.css`와 `script.js` 경로에 `v0.4.2.1` cache busting query를 추가했습니다.

### v0.4.2

- 감자 캐릭터를 더 포메라니안다운 전신 실루엣의 SVG 구조로 재설계했습니다.
- 실제 감자 reference와 `workflow-pom-style.png`를 참고해 아이보리 털, 진한 눈, 작은 코, 분홍 혀, 말린 꼬리 표현을 보완했습니다.
- 쿠션을 약하게 처리하고 러그 대비를 조정해 캐릭터가 배경에서 더 잘 분리되도록 했습니다.
- reference 이미지가 공개되지 않도록 `assets/reference/`를 `.gitignore`에 추가했습니다.

### v0.4.1

- GitHub Pages 배포 반영 여부를 확인하기 위한 `v0.4.1` 버전 표시를 추가했습니다.
- 감자 캐릭터의 둥근 머리, 작은 귀, 복슬복슬한 외곽, 말린 꼬리 표현을 보완했습니다.
- 감자 캐릭터가 러그와 쿠션에서 더 잘 구분되도록 외곽선, 그림자, 색상 대비를 조정했습니다.

## 실행 방법

1. 이 저장소를 로컬에 내려받습니다.
2. `index.html` 파일을 브라우저에서 엽니다.
3. 화면에 `v0.4.4`, `Potato’s Day`, `감자의 하루`, `감자의 집` 문구가 보이는지 확인합니다.
4. `assets/characters/gamja-base.png` 투명 배경 이미지가 홈 화면의 메인 감자 캐릭터로 보이는지 확인합니다.
5. 감자 주변에 체크무늬 사각형 배경이 없고 러그 위 또는 러그 근처에 자연스럽게 배치되었는지 확인합니다.
6. 시작 패널의 작은 아이콘 버튼을 클릭해 워크플로우 모달이 열리는지 확인합니다.
7. 닫기 버튼 또는 모달 바깥 배경을 클릭해 모달이 닫히는지 확인합니다.
8. 브라우저 개발자 도구 Console에서 `Potato’s Day v0.4.4` 메시지를 확인합니다.
